How to Make a Custom polygon Map for Tableau


Here I present a rather simple and straightfoward way to create a custom polygon Map in Tableau.
This unorthodox approach can be applied to any shape and will dispense with the time consuming step of getting all the coordinates and placing them in a spreadsheet. This will be achieved by using the free software Google Earth Pro.

In broadlines we are going to:

  • ■ Import an image to Google Earth.
  • ■ Draw the desired polygons on the image.
  • ■ Export the geografical data.
  • ■ Import the geografical data into Tableau.




Importing Image to the Google Earth

It is not entirely necessary, but very helpful to have a image to base your polygon map on. Here we will use an image of a skateboarder from a project for locating the most common body injuries for this sport.
On the Google Earth menu select "ADD>IMAGE OVERLAY". After that resize your image and position it, preferably on the ocean.



ㅤㅤ




Drawing polygons

Now wer are going to start drawing polygons. It is important to plan beforehand what are the areas that are going to be drawn as well as how we want to group them.
On the Google Earth menu select "Add polygon". A new window will pop-up and you will be able to start drawing the polygon.





With the cursor and the image as background, start clicking on the contour of one of the polygons.



With the polygon drawn, select a name and a description for the polygon.
WHEN IMPORTING THE SPACIAL DATA ON TABLEAU THOSE INFORMATIONS WILL CONSTITUTE ITS OWN COLUMNS, so select carefully.
Keep repeating the last steps until you build all the desired polygons.







Exporting

After drawing all the polygons, archive them together in a single file using the toolbar from Google Earth.
Right-click over the file, select Save Place As and save it as a .kmz file



ㅤㅤ




Importing to Tableau

After drawing all the polygons, archive them together in a single file using the toolbar from Google Earth.
Right-click over the file, select Save Place As and save it as a .kmz file. After importing the file, just treat it the same way you would treat map spacial data.







Congratulations. From now on, the next steps are up to you, Chart Sculptor.
Let's have a final look at a comparison between the inicial image and the final polygons.
