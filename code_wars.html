Code Wars

This is a continuous project that uses the Code Wars Platform for learning and practicing codings languages.

The Code Wars is a collaborative platform where coding challenges are proposed and solved by users in structure fashion. The focus goes from fundamentals until complex problem solving.

In this particular project, I register all solved problems in a Github Repository, where problems can be not only reviewed, but also recycled when similar real problems arise.

Currently 38 SQL chanllenges solved.

GOAL

Improve and maintain Coding skills in SQL.

RESULTS

38 SQL coding challenges solved and registered in a repository for reference.

HIGHLIGHTS

The platform grant access of others users' solutions, allowing for peer improvement through comparisons.
